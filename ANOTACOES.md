## Controles

- Adicionar uma opcao para jogar usando analogico virtual para mover a nave.
- Adicionar um botao `SHOT` na tela para atirar manualmente.
- Manter a opcao atual de tiro automatico para quem preferir.
- Permitir escolher o modo de controle nas configuracoes ou no menu inicial.
- Colocar isso tudo nas configuraçoes do jogo (possibilidade de escolher modo de controle).
